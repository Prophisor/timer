# Timer App

Eine einfache Desktop-Anwendung zur Zeitmessung, entwickelt in C#.

## Funktionen

- **Countdown-Timer:** Einstellen und Starten eines Countdowns.
- **Ressourcenverwaltung:** Unterstützung für lokalisierte Zeichenfolgen und Ressourcen über die `Resources.Designer.cs`.
- **Einfache Benutzeroberfläche:** Intuitive Bedienung für die tägliche Nutzung.

## Voraussetzungen

- **.NET Framework / .NET SDK:** (Basierend auf den Projektdateien, z.B. .NET 4.8 oder höher).
- **Visual Studio:** Empfohlen wird Visual Studio 2022 für die beste Kompatibilität.

## Installation & Ausführung

1. Klone das Repository oder lade den Quellcode herunter.
2. Öffne die Projektdatei (`.csproj`) oder die Solution (`.sln`) in Visual Studio.
3. Erstelle das Projekt mit `Strg + Shift + B`.
4. Starte die Anwendung mit `F5`.

## Projektstruktur

- `Properties/`: Enthält Konfigurationsdateien und Ressourcen-Designer.
- `Resources.Designer.cs`: Automatisch generierte Klasse für den Zugriff auf lokalisierte Ressourcen.
